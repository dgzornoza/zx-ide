# NOTICE

## Development Container

This project uses a Docker development container with pre-configured toolchains for ZX Spectrum development.

**Container Image:** `dgzornoza/zxide-dev:latest`

The development container includes the following toolchains:

- **z88dk** - C/ASM cross-compiler for Z80 systems
- **sjasmplus** - Z80 cross-assembler
- **pasmo** - Z80 cross-assembler

These tools are provided within the container image and are not part of this project's source code. For more information about the toolchains, please refer to their respective documentation and licenses.
